# 💎 AI-Powered Customer Lifetime Value (CLV) Prediction System

---

## 📌 Overview

This project is an end-to-end **Customer Lifetime Value (CLV) Prediction System** built using Machine Learning and deployed with Streamlit. It analyzes customer transaction data, segments users based on behavior, and predicts future revenue potential.

The system combines **RFM analysis + ML models + interactive dashboards** to deliver actionable business insights.

---

## 🎯 Key Features

- 📊 Interactive Streamlit dashboard
- 🤖 Multiple ML models for CLV prediction:
  - Linear Regression
  - Ridge & Lasso Regression
  - Random Forest
  - Gradient Boosting
- 🧩 Customer segmentation (High / Medium / Low value)
- 📈 RFM (Recency, Frequency, Monetary) analysis
- 🔍 Feature importance & correlation analysis
- 📋 Customer-level insights & filtering
- 🧾 Raw transaction exploration
- ⚡ Real-time CLV prediction tool

---

## 🛠️ Tech Stack

- **Frontend/UI**: Streamlit  
- **Backend**: Python  
- **Libraries**: pandas, numpy, scikit-learn, matplotlib, seaborn  
- **Deployment**: Streamlit Cloud
